# React Boilerplate
Development stack consists of React, Babel, Express, Jest, Webpack.

## Setup
```
$ npm install
$ npm run build
$ npm start
```
Open http://localhost:3000/

## Tests
Unit Testing with Jest
```
$ npm test
```
